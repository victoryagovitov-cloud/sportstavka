🚀 ОСНОВНЫЕ КОМПОНЕНТЫ:

1. 🎯 ГЛАВНЫЙ ПЛАНИРОВЩИК (main.py)
   - Управление циклами анализа каждые 45 минут
   - Координация всех модулей
   - Обработка ошибок и восстановление

2. 🕷️ СИСТЕМА СБОРА ДАННЫХ (scrapers/)
   - Парсинг scores24.live для всех видов спорта
   - Глубокий сбор данных со всех вкладок
   - Умная фильтрация по критериям качества

3. 🤖 AI АНАЛИЗ (ai_analyzer/)
   - Интеграция с Claude AI
   - Профессиональный анализ собранных данных
   - Формирование рекомендаций для ставок

4. 📱 TELEGRAM ИНТЕГРАЦИЯ (telegram_bot/)
   - Автоматическая публикация в канал @TrueLiveBet
   - Красивое форматирование отчетов
   - Обработка ошибок отправки

5. 🛠️ УТИЛИТЫ И ЛОГИРОВАНИЕ (utils/)
   - Комплексная система логирования
   - Работа с московским временем
   - Вспомогательные функции
